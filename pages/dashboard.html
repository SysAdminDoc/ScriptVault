Operational Notes

                

                    
Install scripts: Navigate to any .user.js URL, use Find Scripts, or drag files onto this dashboard.

                    
Manage scripts: Toggle on/off, click name to edit, right-click update icon to force-update.

                    
Templates: Click + New to pick from 6 starter templates.

                    
Cloud sync: Settings tab, Sync section. Supports WebDAV, Google Drive, Dropbox, and OneDrive.

                    
Bulk actions: Select scripts with checkboxes, Shift+click for range, then use the action dropdown.

Trusted Types And MAIN-World Scripts

                

                    
Default path: ScriptVault runs scripts in USER_SCRIPT world, which is separate from page Trusted Types policy.

                    
MAIN-world path: Scripts that use page context or unsafeWindow must obey the page policy for raw innerHTML, script URLs, and inline handlers.

                    
Safer DOM writes: Prefer textContent, append, createElement, and GM_addElement with attributes.

                    
TrustedHTML: Use a target-site approved policy when the page requires it; do not create broad passthrough policies to bypass CSP.
