Quick Actions
Operational Notes
Install scripts: Navigate to any .user.js URL, use Find Scripts, or drag files onto this dashboard.
Manage scripts: Toggle on/off, click name to edit, right-click update icon to force-update.
Templates: Click + New to pick from 6 starter templates.
Cloud sync: Settings tab → Sync section. Supports WebDAV, Google Drive, Dropbox, OneDrive.
Bulk actions: Select scripts with checkboxes (Shift+click for range), then use the action dropdown.
